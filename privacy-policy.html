AvtoFair

Privacy Policy

Last updated: March 24, 2026

AvtoFair helps users evaluate used-car listings on Avto.net and mobile.de by analyzing supported listing pages, comparing them with similar listings, and allowing users to save listings and monitor price changes.

Data processed

AvtoFair reads content from supported listing pages only as needed to provide its core features. This may include listing title, price, mileage, year, equipment, description text, images, seller details, location, and listing URL.

How data is used

  • To analyze the current car listing.
  • To compare the listing with similar listings on supported marketplaces.
  • To estimate a fair price and generate a deal score.
  • To save listings to the user’s watchlist.
  • To detect and notify the user about price drops on saved listings.

Storage

AvtoFair stores settings, watchlist items, panel state, listing URLs, and analysis results locally in the browser using Chrome storage.

Data sharing

AvtoFair does not sell user data and does not transfer user data to third parties for advertising, profiling, or unrelated purposes. AvtoFair does not send listing data to developer-operated servers.

Network access

AvtoFair may request pages from supported marketplaces, including Avto.net and mobile.de, in order to compare listings and refresh saved watchlist items.

Notifications

If enabled by the user, AvtoFair uses browser notifications to alert the user when a saved listing drops in price.

User control

Users can remove saved watchlist items at any time, disable notifications, and uninstall the extension to stop all future data processing by the extension.
